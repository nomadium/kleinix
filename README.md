# riscv-hello-uart
Minimal bare-metal RISC-V assembly code with UART output for execution in QEMU

## Requirements
### Tools:
- riscv64-unknown-elf-gcc
- riscv64-unknown-elf-ld
- riscv64-unknown-elf-objcopy

### Building:
Make

### Execution:
qemu-system-riscv64

## Building
make all

## Running
make run
